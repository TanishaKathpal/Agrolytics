About the Project

Agrolytics is a crop recommendation system powered by machine learning and in-depth data analysis. We processed and analyzed datasets containing soil nutrients, weather data, and historical crop performance. Feature selection, model evaluation, and visualization helped us build a highly accurate Extra Trees Regressor model to predict the best-suited crop,as well as predicting the yield output. With this model integrated using Flask and a sleek frontend, Agrolytics serves as a practical tool for smarter, data-backed agricultural decisions.
